# Assessment_SAfG
Simple 2D Game Engine built on top of a DirectX wrapper.

## Documentation.
Code is fully commented.

## Game
#### Kill all zombies in the scene.
#### Collect power-ups thorughout the map to deal more damage.
#### Game consists of 2 levels.

## Control
#### WASD - Move.
#### Spacebar - Shoot.
